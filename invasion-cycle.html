INVASION CYCLE

A solo project I did for college. It's a mobile game taking inspiration on 'Angry Birds' mechanics, but with a darker atmosphere. My main focus was having solid design principles and documentation, but as a programmer I wanted to implement my idea, so I prototyped the game as a whole. It's really short and unpolished, but it showed what my idea consisted of.

I did this whole game from scratch. The art, music, coding and design was done by me. My biggest take away was probably how all these aspects of game development intertwine and how to put it all together, it helped me to further my knowledge of implementation in game engines, in this case Unity.

As I stated earlier, the game was originally made for Android, but later I ported it to PC.
