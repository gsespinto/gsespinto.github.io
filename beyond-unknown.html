BEYOND UNKNOWN

In "Beyond Unknown", you follow a wierd being stuck in a looping world. It has to adapt itself to overcome it and become free.

This project was made in a week by a group of three people. Among them I was the programmer and delt with the engine (importing assets, placing level design, post processing...). The game was made using Unity 2D and C#. It was an interesting project because my other colleagues hadn't delt in game development beforehand, and it brought a lot of creativity to the experience.

One of the things I enjoyed the most working on this project was dealing with the never ending loop mechanic and how the narrative queues were triggered.
