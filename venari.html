PROJECT M

'Project M', later known as 'Venari', is a third person Action RPG game in which you play as bounty hunters that face monsters in a post-apocalyptic setting.
The project was developed by a group of seven members, from which I was the gameplay and UI programmer. My delivery consisted of the Code Release milestone of the project, in which the code base for the project was completed.
The tools I utilized in this project were C++, Blueprints and the Unreal Engine, in combination with GitHub for the repository management.
I worked on systems such as melee combat, dialogue trees and inventory management. I documented my process, having in mind that the project would still be undergoing development after this delivery.
Personally, I am not finished with the project. My goal now is to take the code base I build for 'Project M' and make a gameplay demo with some free assets, showcasing more specifically my part in this project in a more appealing/interesting way.
